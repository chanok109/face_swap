วิธีเพิ่มรูปตัวละครให้เด็กๆ เลือก
====================================

1. หารูปที่มีใบหน้าคนชัดเจน หันหน้าเข้ากล้องตรงๆ แสงสว่างพอ (เช่น รูปนักบินอวกาศ, ซูเปอร์ฮีโร่, เจ้าหญิง, โจรสลัด ฯลฯ)
   - ต้องเป็นรูปที่คุณมีสิทธิ์ใช้งานนะ (รูปที่ซื้อลิขสิทธิ์ถูกต้อง หรือรูปฟรีจากแหล่งอนุญาตใช้งานเชิงพาณิชย์)
2. ตั้งชื่อไฟล์เป็นภาษาอังกฤษ ใช้ - หรือ _ แทนช่องว่าง เช่น
   - astronaut.jpg      -> จะแสดงชื่อว่า "Astronaut"
   - super-hero.png     -> จะแสดงชื่อว่า "Super Hero"
   - princess_castle.webp -> จะแสดงชื่อว่า "Princess Castle"
3. รองรับไฟล์นามสกุล .jpg .jpeg .png .webp เท่านั้น
4. วางไฟล์ลงในโฟลเดอร์นี้ (public/gallery/) แล้ว refresh หน้าเว็บ ระบบจะโหลดขึ้นมาให้เลือกอัตโนมัติ ไม่ต้องแก้โค้ดใดๆ

แนะนำ: ใช้รูปขนาดไม่เกิน 1500px ต่อด้าน จะโหลดเร็วขึ้น
